# Prérequis
1) Tomcat installé

# Installation
1) Utiliser maven pour installer les dépendances.


# Configuration
1) Créer à la racine un fichier nommé `.env` avec vos infos à partir de l'exemple `.env.example`.
# Utilisation
Exécuter la classe AirportMadnessApplication pour lancer
le serveur web.
